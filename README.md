LLA – Launguage Learning Aplication	
==========

MyLaunguageLearningAplication is a Xamarin aplication written in C# to fast learning new words in a foregin language. The first version of the application is limited only to adding new words with translation and writing it to SQLite base. Aplication is not working now.


Supported operating systems
---------------------------
- Android 6.0 and newer
